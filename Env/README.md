# Env - 전역 환경 설정 패키지

이 패키지는 monorepo의 모든 서브 패키지에서 공유되는 의존성과 환경 설정을 관리합니다.

## 목적

- 모든 패키지에서 동일한 버전의 의존성 사용
- 중복 설치 방지
- 의존성 관리 단순화

## 사용 방법

각 서브 패키지(server, browser 등)의 `package.json`에서 다음과 같이 참조합니다:

```json
{
  "dependencies": {
    "@workwork/env": "file:../Env"
  }
}
```

## 의존성 추가

새로운 의존성을 추가하려면 이 폴더의 `package.json`을 수정하고, 
루트에서 `npm install`을 실행합니다.

## 주의사항

- 이 패키지는 직접 실행되지 않습니다
- 모든 의존성은 여기서 중앙 관리됩니다
- 버전 업데이트 시 모든 서브 패키지에 영향을 미칩니다
